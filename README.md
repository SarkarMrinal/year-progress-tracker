# Year Progress Tracker

A beautiful, single-file HTML application that visualizes the passage of time. Track your year, month, week, and day progress with a stunning, modern interface featuring glassmorphism design, ambient lighting effects, and GitHub contribution integration. **Originally designed for personal use to be set as a browser homepage.**

## ‚ú® Features

- **Real-time Progress Tracking**: Visual bars and percentages for:
  - Year (with day count and days remaining)
  - Month
  - Week
  - Day
- **GitHub Contributions**: View your GitHub contribution graph directly on the dashboard.
- **Dynamic Visuals**:
  - Glassmorphic card design
  - Ambient background orbs and particle effects
  - "Fire" glow effects on progress bars
  - Animated sparkles and floating elements
- **Interactive UI**:
  - **Nightlight Mode**: Toggle a warm overlay for late-night viewing.
  - **AFK Indicator**: detects when you are away from the keyboard.
  - **Settings**: Customizable user name and GitHub username.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.

## üöÄ Getting Started

Since this is a standalone HTML file, there's no build process required!

1.  **Download**: Clone this repository or simply download the `year-progress-tracker.html` file.
2.  **Open**: Double-click the file to open it in your default web browser.
3.  **Setup**:
    - On first load, you'll be welcomed by a setup modal.
    - Enter your name and GitHub username to personalize the dashboard.
    - These settings are saved locally in your browser.

## üõ†Ô∏è Customization

The project uses [Tailwind CSS](https://tailwindcss.com/) (via CDN) for styling, making it easy to tweak.

- **Colors**: The `tailwind.config` script inside the `<head>` tag defines the color palette. You can modify the `accent`, `background`, and `surface` colors to match your preference.
- **Fonts**: Uses Google Fonts (Inter, JetBrains Mono, Space Grotesk).
- **Background**: The background features a layered noise texture and radial gradients which can be adjusted in the CSS.
